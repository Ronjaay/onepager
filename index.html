Animations in CSS

Animations = advanced programming languages? Actually you can create a lot of very cool animations just using css!

How it works

Step 1: Create the Element

Start by creating a simple HTML element, such as a square div with basic styles.

Step 2: Style the Element

Apply CSS styles to shape and color your element. This step adds the visual appearance.

Step 3: Add Keyframes

Define keyframes to specify the animation's starting and ending points, and how it transitions.

Step 4: Enjoy the Animation

Apply the animation to your element and watch it come to life!

We first create our square element. Then we will give the div a class named for example "pink-square". This way we can style our element in the next step.

Step 3: Keyframes

By adding keyframes we define the behavior of our animation. from defines the start and to the end position of our animation. Here we set it to move 100px to the right on the x-axis (translateX(100px)).

Let's go! This is what your code could look like: 

@keyframes move { from { transform: translateX(0); } to { transform: translateX(100px); } }

Step 4: Apply the Animation

Almost done! All we have to do is to give our animation some finishing touches. We will set the duration of the animation to 4 seconds and let it run infinitely while using alternate to switch directions with each cycle. Notice that the name of the animation (here it is move) has to be the same as the one you used in your keyframes! Go back to your code in step 2 and add definition to the .pink-square class!
Let's go! This is what the animation definition could look like: 

.pink-square { animation: move 4s infinite alternate; }
